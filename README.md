# CloudedAI - Next.js | TypeScript | Prisma | Pinecone | MongoDB - Full Stack Project

## Overview of Project
AI integrated note-taking app with Chatbot assistant.


## Resources
  - OpenAI API:
    - ChatGPT 3.5
  - Next.js
  - Tailwind.css
  - TypeScript
  - Vercel AI SDK
  - Pinecone
  - Prisma
  - Clerk
  - MongoDB
  - Shadcn UI
  - Heroku
